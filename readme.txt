== Description ==

Contact form master is a user-friendly form builder plugin that allows to create many contact forms with different fields.

** Contact form master - Fields: **

* text field
* number field
* text area field
* Address field
* select box field

** form - Submit options: **

* from Email
* Contact form to Email
* Email Subject
* Contact form Message

If you think that you have found a bug in Contact form master plugin or have any questions, please feel free to contact us at <b>edmon.parker@gmail.com</b>.
